# 🧮 智能计算器

一个功能丰富的在线计算器，支持基础运算、科学计算和房贷计算。

## 功能特性

- ✨ **基础计算器**：四则运算、小数点、连续计算、键盘支持
- 🔬 **科学计算器**：三角函数、对数、指数、括号、角度/弧度切换
- 🏠 **房贷计算器**：商业贷款、公积金贷款、等额本息/等额本金计算

## 技术栈

- HTML5 + CSS3 + 原生 JavaScript (ES6+)
- 无外部依赖
- 响应式设计
- 现代简约 UI（紫色渐变主题）

## 本地运行

直接在浏览器中打开 `index.html` 即可。
